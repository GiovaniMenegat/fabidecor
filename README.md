# FabiDecor

Landing page for an independent professional painter based in Auckland, New Zealand.

## Services

- **Painting** — general interior/exterior painting (flagship service)
- **Maintenance** — general property maintenance
- **House washing**
- **Driveway cleaning**
- **Moving out / end-of-tenancy painting**
- **Windows cleaning**

See [AGENTS.md](./AGENTS.md) for full project context (locale, audience, and how this list should drive the site's content).

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Stack

Next.js (App Router) · React · TypeScript · Tailwind CSS v4 · ESLint
